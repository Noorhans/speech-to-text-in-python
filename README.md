🎤 Speech-to-Text Desktop Application (Python)
📌 Overview
This is a Python-based Speech-to-Text application with a simple graphical user interface.
It allows users to record audio through their microphone and convert it into text using a speech‑recognition engine.
The project includes a custom application icon and a clean folder structure following software‑engineering best practices.
✨ Features
• 	Real-time speech recognition
• 	Simple and user-friendly UI
• 	Custom application icon
• 	Lightweight and easy to run
